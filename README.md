# Unc Studios Website

Simple Next.js website for Unc Studios, including:

- A homepage for studio information
- A Privacy Policy page for Hull Keeper
- A `public/app-ads.txt` file for AdMob verification

## Development

```bash
npm install
npm run dev
```

## Deployment

Deploy to Vercel or any standard Next.js hosting provider.

Before deployment, replace the placeholder publisher ID in `public/app-ads.txt`:

```txt
pub-REPLACE_WITH_MY_ADMOB_PUBLISHER_ID
```
